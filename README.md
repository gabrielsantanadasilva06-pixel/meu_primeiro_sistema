# 🎓 Sistema Acadêmico em Python (Flask)

Este projeto é um **Sistema Acadêmico Web** desenvolvido em **Python utilizando Flask**, com interface em **HTML (Jinja2)** e armazenamento de dados em **JSON**.

O objetivo do projeto é praticar **desenvolvimento backend, estrutura de sistemas administrativos e manipulação de dados**, simulando funcionalidades de um sistema acadêmico real.

---

## 🚀 Funcionalidades

✔ Cadastro de alunos  
✔ Listagem de alunos  
✔ Cálculo automático de média  
✔ Classificação de situação do aluno  
- Aprovado  
- Recuperação  
- Reprovado  

✔ Exclusão de alunos pela interface web  
✔ Armazenamento de dados em arquivos JSON
